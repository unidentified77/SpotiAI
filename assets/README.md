# Assets Klasörü

Bu klasör uygulama görsellerini içerir. Aşağıdaki dosyalar `app.json` yapılandırmasında referans edilmiştir:

## Gerekli Dosyalar

- **icon.png** - Uygulama ikonu (1024x1024 px önerilir)
- **splash.png** - Splash screen görseli (1284x2778 px önerilir)
- **adaptive-icon.png** - Android adaptive icon (1024x1024 px önerilir)
- **favicon.png** - Web favicon (48x48 px önerilir)

## Notlar

- Tüm görseller PNG formatında olmalıdır
- İkonlar şeffaf arka planlı olabilir
- Splash screen genellikle beyaz veya uygulama temasına uygun bir arka plan rengi içerir
